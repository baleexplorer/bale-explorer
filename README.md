# Bale Explorer

Brand apparel outdoor Indonesia — Built for the wild, made for everyday.

## Tech Stack

- Next.js 14 (App Router, Server Components)
- TypeScript (strict mode)
- Tailwind CSS
- Framer Motion
- Lucide React

## Getting Started

\`\`\`bash
npm install
npm run dev
\`\`\`

Visit http://localhost:3000

## Deployment

Push ke GitHub repo → connect ke Vercel. Tidak ada static export (SSR required untuk blog dan API routes).
